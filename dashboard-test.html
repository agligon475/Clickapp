Categorías de productos
Administrá las categorías disponibles en tu tienda. Los productos se asignan a una categoría al crear o editar. Tabla categories en Supabase →
Cargando categorías...
Para agregar nuevas categorías:
1. Abrí Supabase Dashboard
2. Tabla categories → New row
3. Ingresá: name (ej: "Herramientas"), emoji (ej: "🔧"), active (true)
4. Recargá esta página
